Página inicial

Essa imagem mostra a página principal do projeto de Lista de Tarefas em PHP e MySQL, com um container de lista de tarefas e um botão para adicionar novas tarefas.

Adicionar tarefa

A segunda imagem mostra o modal que surge após o usuário clicar no botão "Adicionar Tarefa". Ele oferece um espaço para o usuário inserir a tarefa por meio de um campo de texto e um botão "Adicionar".

Após a adição da tarefa

A imagem mostra a lista de tarefas atualizada com a adição de uma nova tarefa pelo usuário. Com esse sistema simples e eficaz, o usuário pode manter uma lista organizada de tarefas e aumentar sua produtividade.

Lista de tarefas V.1

Este projeto foi desenvolvido com o objetivo de aprimorar minhas habilidades em PHP & MySQL. Ele permite que o usuário adicione e remova itens, com foco na interação com o banco de dados.
